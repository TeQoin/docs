# WEB3 methods

`web3_*` methods are helper calls that clients often use for basic metadata and hashing helpers.

## Common methods

- `web3_clientVersion`
- `web3_sha3`

These are useful for diagnostics and low-level tooling, but most app teams spend more time in `eth_*` methods.
